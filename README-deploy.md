# Deploy: Dawn Brew Coming Soon Page

## Deploy to Vercel
1. Push code to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click "New Project" and import the repository
4. Confirm settings:
   - Framework: Auto-detected (static HTML)
   - Root directory: project root
   - Build command: none (inferred as static site)
   - Output directory: project root
5. Click "Deploy"

## Environment Variables
None required.

## First-time setup
None required. This is a static site with no database, authentication, or external services.
